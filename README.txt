#projet_ai
This code contains several poker-playing computer programs.

To use the agents run the command:

	python poker.py

Then choose the mode of playing for both player (via terminal, the code will ask you):

If you want to play against another player, choose mode 1 for one of the player and the corresponding mode for the other player.

If you want to make 2 agents play each other, choose the corresponding mode for each player.

If you want to modify some game constants, like the initial number of chips, 
the value of the blind, the value of the Bet, or the number of game to play:

Go in the poker.py file, and modify the constants values, starting at line 48
